# Patch 0.11.8
> Small follow-up patch with fixes and balance changes

## Features:
- Library now has 2 extra filters: *In Shop* and *Discovered*. The *In Shop* filter is automatically toggled when accessing the library popup by right-clicking an encounter

## Units:
- Kudzu: Nerfed Cooldown from 3s -> 4s, Health Buff from +5 -> +2
- Stone Column: Is now added to the game (The unit spawned by Jefferson)
- Mecha Dragon: Nerfed Damage from 90 -> 80

## Heroes:
- ZapZap:
  - Thriving Trees quest: From needing 5 trees -> 4
  - Overgrowth: From needing 6 Nature units -> 5
- Ambrose:
  - Infrastructure quest: From getting 1 powerline -> 2 powerlines

## Bugs:
- In-game bug report would sometimes close on click
- Island stats UI blocking the unit UI

## Other:
- In-game bug report will now send extra server related logs to us to fix an ongoing matchmaking issue
- Combat-end damage animation will now speed up if opponent has more units
