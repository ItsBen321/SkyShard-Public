# Patch 0.11.2
> Regular patch fixing stuff and introducing some small new things.

## Units:
- Materia Contrarium: Now splits the Damage only between the targets that can buff their Damage (before this change, the damage would also try to get put on non-weapons, lowering the total amount that could go on a single weapon)
- Mecha Dragon Rework: Now triggers whenever it gets a speed buff!
- Weather Machine: Damage buff from +1 to +0.5
- Scarecrow: Now only targets cooldown unit
- EMP mine: Now only targets enemy cooldown units
- New art:
  - Red Sun Shrine (changes when it takes damage)
  - Mecha Egg
  - Mecha Dragon
  - Magnus the Miniscule
  - EMP mine
+ New sound effects:
  + Weather Machine
  + Warden's Flame
  + Toy Turret
  + Soul Slinger
  + Rusty Sentry
  + Myrria Cursebinder
  + Mecha Dragon
  + Forest Spirit
  + Firepot Trebuchet
  + Entombed Terror
  + Toy Production
  + Mechanic
  + Graveyard

## Heroes:
- Zapzap: now has increased chance for Nature shops
- Auris: now has increased chance for Novus shops
- Ambrose: now has increased chance for Town shops

## Bugs:
- Clicking "Tips" on the unit UI in the library would close the library
- Dragging enchantment from the inventory didn't always get put on units
- Altar of Decima and Mini Black Hole can no longer resummon units that are permanently destroyed
- Refractor would not register some units properly

## Other:
- Quest select screen now has a darker background
- Selecting the "random hero" option now needs confirmation
- Custom mouse cursors are now slightly bigger
- Changed button sound effects
- Vile totem info text changed to "Reflects back to the *source*"
- Transmute can now only appear once every round
